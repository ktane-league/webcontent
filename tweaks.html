Tweaks settings

The holdable demands Tweaks to be loaded and enabled, but you have to make sure it has the correct settings set:

Setting name Value
Better Case Picker Disabled
Bomb HUD Enabled
Show Edgework Disabled
Disable Advantageous Features Disabled
Show Tips Disabled
Mode Normal
Mission Seed -1
Case Generator Disabled
Module Tweaks Disabled


Values of the settings that are not listed in the table above doesn't matter.
